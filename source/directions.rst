Directions and Advice
=====================

The four basic cardinal directions are:

+---------------+--------------+-------------+
|               | to the North |             |
+---------------+--------------+-------------+
|               | au nord      |             |
+---------------+--------------+-------------+
| to the West   |              | to the East |
+---------------+--------------+-------------+
| à l'ouest     |              | à l'est     |
+---------------+--------------+-------------+
|               | to the South |             |
+---------------+--------------+-------------+
|               | au sud       |             |
+---------------+--------------+-------------+
